e-dumper

A multi-mode study tool for practicing quizzes, learning vocabulary, and reviewing text content.

Quick Start

  1. Open e-dumper.html in any modern web browser
  2. Select a mode from the dropdown (Quiz, Word List, or One Liner Q)
  3. Import your study material and start learning

Features

Quiz Mode

  • Import questions from text files
  • Two study modes:
    • Study Mode: Highlights correct answer or shows answer on button click
    • Exam Mode: No answer revealed until you click "End Exam"
  • Flag questions for later review
  • Track correct/incorrect answers
  • Export wrong answers or flagged questions

Word List Mode

  • Import word lists (CSV format: word,meaning)
  • Flashcard-style display
  • Shuffle: Randomize word order (checkbox persists across sessions)
  • Swap: Reverse mode - show meaning as the main word
  • Flag words for review
  • Export flagged words

One Liner Q Mode

  • Import text files
  • Display one line at a time
  • Shuffle lines option
  • Navigate with Previous/Next

File Formats

Quiz Format (.txt)

Question 1: What is 2+2?
A) 3
B) 4
C) 5
D) 6
Answer: B
  • Questions: Question N:, N., N), or Question:
  • Options: A), A., A prefix
  • Answers: Answer: X or Answer: X,Y (multi-select)

Word List Format (.csv, .txt)

apple,a fruit
banana,a yellow fruit
cherry,a red fruit
  • Format: word,meaning or word|meaning

One Liner Format (.txt)

Line 1 content
Line 2 content
Line 3 content

Data Persistence

Your progress is saved automatically to localStorage:

  • Current position in each mode
  • Flagged items
  • Shuffle and Swap checkbox states
  • Quiz answers and scores

Running the App

Simply open e-dumper.html in a browser. No server required.

# Using Python
python -m http.server 8000 -d e-dumper/

# Using npx
npx serve e-dumper/

Then open http://localhost:8000/e-dumper.html

Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
